# 🚀 NLP Foundational Projects

This repository contains a collection of **foundational Natural Language Processing (NLP)** projects implemented using deep learning techniques. It is designed to build strong intuition from **basic text preprocessing → embeddings → sequence models → text generation → conversational AI**.

---

## 📂 Project Structure

### 🧩 1. Tokenization Techniques

**File:** `collection_of_different_tokenizers.ipynb`

* Implemented and compared different tokenization strategies
* Word-level, character-level, and subword tokenization
* Understanding vocabulary creation and preprocessing pipelines

---

### 🔤 2. Word Embeddings (Basics)

**File:** `fun_with_word_embeddings_.ipynb`

* Introduction to dense vector representations
* Exploring semantic relationships between words
* Visualizing embeddings and similarity concepts

---

### 🌍 3. GloVe Embeddings for Multi-label Classification

**File:** `using_glove_embeddings_for_multi_label_classification.ipynb`

* Using pre-trained **GloVe embeddings**
* Building a **multi-label text classification model**
* Improving performance using transfer learning

---

### 🔁 4. RNN for Next Word Prediction

**File:** `using_RNN_to_predicting_next_word.ipynb`

* Implemented Recurrent Neural Networks (RNN)
* Sequence modeling for language prediction
* Predicting the next word given a sequence

---

### 🧠 5. LSTM for Text Generation

**File:** `using_LSTM_for_text_generation.ipynb`

* Using LSTM to overcome vanishing gradient problem
* Training a generative model for text
* Generating human-like sequences

---

### 💬 6. Seq2Seq Model for Conversational AI

**File:** `using_seq2seq_model_for_building_conversational_model.ipynb`

* Encoder-Decoder architecture
* Sequence-to-sequence learning
* Building a basic chatbot system

---
